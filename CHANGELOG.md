# Changelog

All notable changes to TeleAgent will be documented in this file.

The format follows Keep a Changelog, and this project uses semantic versioning.

## [0.1.0] - 2026-06-10

### Added

- PTY wrapper for interactive CLI agents.
- Regex-based auto-reply rules.
- Telegram bridge for forwarding user messages to the wrapped CLI.
- Mobile-friendly output cleanup, summaries, history export, and debug mode.
- Terminal menu mirroring for model/session/approval prompts.
- Optional Codex rollout and Kimi wire/context output sources.
